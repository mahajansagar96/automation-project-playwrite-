# Sample Automation Framework (Playwright)

## 🚀 Overview

This project is an end-to-end automation framework built using Playwright. It automates a campaign management system including login, campaign creation, and validation.

## 🧰 Tech Stack

* Playwright (JavaScript)
* Node.js
* Axios (API Testing)

## 📂 Framework Features

* Page Object Model (POM)
* API + UI Testing
* Environment Configuration
* Reusable Components
* HTML Reporting

## 🧪 Test Scenarios

* Login functionality
* Campaign creation
* Campaign validation in UI
* API validation of campaigns

## ▶️ How to Run

```bash
npm install
npx playwright test
```

## 📊 Reports

```bash
npx playwright show-report
```
